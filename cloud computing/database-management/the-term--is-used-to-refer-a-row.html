The term ____ is used to refer a row

(A) ** Record

(B) Primary key

(C) Foreign key

(D) Field

EXPLANATIONS BELOW

Concept note-1: -A row, or record, is also known as a tuple. The columns in a table is a field and is also referred to as an attribute. You can also think of it this way: an attribute is used to define the record and a record contains a set of attributes.

Concept note-2: -The table name and column names are used to help in interpreting the meaning of values in each row. Each row of a table is called a data record.

Concept note-3: -However, we refer to these rows and a row as “Tuples” in DBMS. The relational model uses a unique key to organize data into at least one table of rows and columns. The rows are referred to as tuples or records, and the columns are referred to as attributes.

Concept note-4: -The term tuple is used to refer to a record in a table. The table helps the user to keep their data in a format that contains rows and columns. And various terms are used to refer to the column, rows, etc. The attribute terms denote the feature or characteristics of data in the table.

Concept note-5: -The term is used to refer to a row. D. Instance. Answer» B. Tuple.
