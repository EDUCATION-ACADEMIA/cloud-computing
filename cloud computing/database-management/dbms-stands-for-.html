Dbms stands for ____

(A) ** Database management system

(B) Database management storage

(C) Data base manipulation system

(D) Database manipulation storage

EXPLANATIONS BELOW

Concept note-1: -A database management system (or DBMS) is essentially nothing more than a computerized data-keeping system. Users of the system are given facilities to perform several kinds of operations on such a system for either manipulation of the data in the database or the management of the database structure itself.

Concept note-2: -A database management system (DBMS) is system software for creating and managing databases. A DBMS makes it possible for end users to create, protect, read, update and delete data in a database.

Concept note-3: -Database management system is a software which is used to manage the database. For example: MySQL, Oracle, etc are a very popular commercial database which is used in different applications.

Concept note-4: -DBMS stands for Database Management System, and RDBMS is the acronym for the Relational Database Management system. In DBMS, the data is stored as a file, whereas in RDBMS, data is stored in the form of tables.

Concept note-5: -hierarchical database systems. network database systems. object-oriented database systems.
