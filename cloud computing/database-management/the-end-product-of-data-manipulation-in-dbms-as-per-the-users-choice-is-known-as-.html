The end product of data manipulation in DBMS as per the user’s choice is known as ____

(A) Record

(B) ** Report

(C) Forms

(D) None

EXPLANATIONS BELOW

Concept note-1: -Answer. The end product of data manipulation in DBMS as per the user’s choice is known as ‘Report’. Report can be obtained as a hard copy or a soft copy. Reports are usually created after making the queries on the database.

Concept note-2: -A DML (data manipulation language) refers to a computer programming language that allows you to add (insert), delete (delete), and alter (update) data in a database. A DML is typically a sublanguage of a larger database language like SQL, with the DML containing some of the language’s operators.

Concept note-3: -Data processing is the collection and manipulation of digital data to produce meaningful information.

Concept note-4: -SQL (Structured Query Language) is a programming language used to communicate with and manipulate databases.
