It is a collection of data that is related to a particular subject or purpose

(A) Field

(B) Row

(C) Column

(D) ** Database

EXPLANATIONS BELOW

Concept note-1: -A database is a collection of information that is related to a particular subject or purpose, such as tracking customer orders or maintaining a music collection.

Concept note-2: -A database is an organized collection of structured information, or data, typically stored electronically in a computer system. A database is usually controlled by a database management system (DBMS).

Concept note-3: -The collection of information stored in the database at a particular moment is called an instance of the database. The overall design of the database is called the database schema.
