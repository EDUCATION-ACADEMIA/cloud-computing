EXPLANATIONS BELOW

Concept note-1: -A column is a collection of cells alligned vertically in a table. A field is an element in which one piece of information is stored, such as the received field. Usually a column in a table contains the values of a single field. However, you can show several fields in a column by using a Formula or a Combination field.

Concept note-2: -Records and Fields in SQL Tables contain rows and columns, where the rows are known as records and the columns are known as fields. A column is a set of data values of a particular type (like numbers or alphabets), one value for each row of the database, for example, Age, Student ID, or Student Name.

Concept note-3: -A table has records (rows) and fields (columns). Fields have different types of data, such as text, numbers, dates, and hyperlinks.

Concept note-4: -The columns in a table is a field and is also referred to as an attribute. You can also think of it this way: an attribute is used to define the record and a record contains a set of attributes.
