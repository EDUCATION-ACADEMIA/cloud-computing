The basic Web Services platform is combination of ____ and ____

(A) XML + HTML

(B) ** XML + HTTP

(C) CSS + HTTP

(D) CSS + JAVA

EXPLANATIONS BELOW

Concept note-1: -Answer: The basic web services platform is the combination of XML and HTTP . Explanation : The full form of Xml is ‘eXtensible Markup Language)’.

Concept note-2: -In other words, if an application can be accessed over a network using a combination of protocols like HTTP, XML, SMTP, or Jabber, then it is a web service.

Concept note-3: -A web service is any piece of software that makes itself available over the internet and uses a standardized XML messaging system. XML is used to encode all communications to a web service. For example, a client invokes a web service by sending an XML message, then waits for a corresponding XML response.

Concept note-4: -XML Web services provide a way to describe their interfaces in enough detail to allow a user to build a client application to talk to them. This description is usually provided in an XML document called a Web Services Description Language (WSDL) document.

Concept note-5: -There are a few central types of web services: XML-RPC, UDDI, SOAP, and REST: XML-RPC (Remote Procedure Call) is the most basic XML protocol to exchange data between a wide variety of devices on a network. It uses HTTP to quickly and easily transfer data and communication other information from client to server.
