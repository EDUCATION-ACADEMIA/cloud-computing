What’s the most popular use case for public cloud computing today?

(A) Website hosting

(B) ** Test and development

(C) Disaster recovery

(D) Business analytics

EXPLANATIONS BELOW

Concept note-1: -Storage. This might be the most common use case for cloud computing.

Concept note-2: -Platform as a Service (PaaS) It offers all the resources needed to enable users to deliver functional and customized applications. PaaS providers and products are typically concerned with software development. It is one of the topmost use cases of cloud computing.

Concept note-3: -Microsoft Azure and Amazon Web Services are examples of a public cloud. We access public cloud resources and services over the internet. So to use a public cloud, we need an internet connection and manage the cloud services and resources through a web portal provided by the cloud service provider.
