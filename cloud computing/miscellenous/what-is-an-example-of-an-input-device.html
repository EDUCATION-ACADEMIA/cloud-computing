What is an example of an input device?

(A) ** USB Drive

(B) USB Port

(C) Motherboard

(D) Power button

EXPLANATIONS BELOW

Concept note-1: -Some examples of input/output devices are as: USB drive: Also known as pen drive or flash stick works as both input device to computer and as an output device. USB drives receive or save data from a computer as an input and it can also send data to a computer or another device.

Concept note-2: -A flash drive is both input and output device, input devices are mouse, keyboard, webcam, scanner it is basically anything that inputs data in the systems.

Concept note-3: -USB flash drive: It is also referred to as a keychain drive, data stick, USB flash drive, thumb drive, memory unit, pen drive. It is a portable storage device that saves or accepts data from a computer (input). It connects to a computer via a USB port and forwards data to a computer or another device (output).

Concept note-4: -A USB flash drive–also known as a USB stick, USB thumb drive or pen drive–is a plug-and-play portable storage device that uses flash memory and is lightweight enough to attach to a keychain. A USB flash drive can be used in place of a compact disc.

Concept note-5: -A USB flash drive (also called a thumb drive in the US, or a memory stick in the UK) is a data storage device that includes flash memory with an integrated USB interface. It is typically removable, rewritable and much smaller than an optical disc.
