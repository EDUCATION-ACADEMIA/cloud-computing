Encryption provides which of the following assurances?

(A) ** confidentiality

(B) ** integrity

(C) ** availability

(D) accessibility

EXPLANATIONS BELOW

Concept note-1: -Data encryption helps ensure the confidentiality, integrity and availability of data at rest and in transit. This is commonly known as the CIA triad: Confidentiality-Data confidentiality focuses on an organizationâ€™s efforts to ensure that data can only be viewed by authorized parties.

Concept note-2: -Authentication and access control techniques are used to provide confidentiality. Data encryption is often used to provide Integrity. Intrusion Detection Systems (IDS) is currently being used to detect intruders, security holes, Trojan Horses and other malicious actions to the system.

Concept note-3: -Encryption is a process that scrambles readable text so it can only be read by the person who has the secret code, or decryption key. It helps provide data security for sensitive information.

Concept note-4: -Encryption involves converting human-readable plaintext into incomprehensible text, which is known as ciphertext. Essentially, this means taking readable data and changing it so that it appears random. Encryption involves using a cryptographic key, a set of mathematical values both the sender and recipient agree on.
