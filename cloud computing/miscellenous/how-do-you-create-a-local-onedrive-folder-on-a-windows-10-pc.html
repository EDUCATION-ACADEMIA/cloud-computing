How do you create a local OneDrive folder on a Windows 10 PC?

(A) Download and install an app from Playstore

(B) Create the folder in File Explorer, then configure it to sync with your OneDrive location

(C) Download and install an app from OneDrive.com

(D) ** A local OneDrive folder is built in

EXPLANATIONS BELOW

Concept note-1: -Select the Start button, search for “OneDrive”, and then open it: In Windows 10, select OneDrive. When OneDrive Setup starts, enter your personal account, or your work or school account, and select Sign in.

Concept note-2: -Want to view your OneDrive files? You can see your locally saved folder and files if you right-click on the OneDrive System Tray icon and select Open folder. This will open a windows in File Explorer that shows all the OneDrive folders and files that are saved directly on your computer.

Concept note-3: -Choose OneDrive Personal or OneDrive for Business. Copy the CID number from the address bar. In Windows Explorer, go to This PC. Click on the Computer tab then “Map network drive". In the Folder field, enter https://d.docs.live.net/ followed by the CID you copied in step 1. More items •14-Sept-2017

Concept note-4: -Click on the “OneDrive” icon in your system tray. Select “Help & Settings” Select “Settings” Select the “Backup” tab, and then select “Manage Backup”. Choose the local folders you would like to backup and sync to your OneDrive folder. Then select, “Start Backup”.

Concept note-5: -Select New > Folder. Type a name for the folder and select Create. Select the files you want and drag them into the folder. 01-Apr-2023
