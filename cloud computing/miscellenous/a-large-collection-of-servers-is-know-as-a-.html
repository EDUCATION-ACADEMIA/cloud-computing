A large collection of servers is know as a ____

(A) Network

(B) Computer System

(C) ** Server Farm

(D) Database

EXPLANATIONS BELOW

Concept note-1: -A server farm or server cluster is a collection of computer servers, usually maintained by an organization to supply server functionality far beyond the capability of a single machine. They often consist of thousands of computers which require a large amount of power to run and to keep cool.

Concept note-2: -What is a server farm? Also known as a server cluster, a server farm is a large number-up to thousands-of servers grouped together to provide better functionality and accessibility. By keeping the servers close together, the manager of a server farm can achieve centralized control and economy of scale.

Concept note-3: -A server farm is a group of computers serving various computing and storage needs from a single location. It is also sometimes called a server cluster. A good example of server farm usage is a high-traffic website. Modern busy sites are generally supported with multiple servers on the backend.

Concept note-4: -A web farm is a group of two or more web servers (or nodes) that host multiple instances of an app. When requests from users arrive to a web farm, a load balancer distributes the requests to the web farmâ€™s nodes.
