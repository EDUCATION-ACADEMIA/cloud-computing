Joe Has read and write access to his own home directory. Joe and Ann are collaborating on a project, and Joe would like to give Ann write access to one particular file in this home directory. Which of the following types of access control would this reflect?

(A) Role-based access control

(B) Rule-based access control

(C) Mandatory access control

(D) ** Discretionary access control

EXPLANATIONS BELOW

Concept note-1: -The Discretionary Access Control, or DAC, model is the least restrictive model compared to the most restrictive MAC model. DAC allows an individual complete control over any objects they own along with the programs associated with those objects.

Concept note-2: -Discretionary Access Control (DAC) – The principle behind DAC is that subjects can determine who has access to their objects. The DAC model takes advantage of using access control lists (ACLs) and capability tables.

Concept note-3: -1. Mandatory Access Control (MAC) The Mandatory Access Control (MAC) model gives only the owner and custodian management of the access controls. This means the end-user has no control over any settings that provide any privileges to anyone.

Concept note-4: -Role-based access control (RBAC) is a method of restricting network access based on the roles of individual users within an enterprise. Organizations use RBAC–also called role-based security–to parse levels of access based on an employee’s roles and responsibilities.
